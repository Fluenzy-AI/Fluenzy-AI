================================================================================
                              DATA FLOW SEQUENCE
================================================================================

1. USER AUTHENTICATION FLOW
   ────────────────────────
   User → Login/Register → NextAuth → Credentials/Google OAuth
      → Verify → Create Session → JWT Token → Client
      → Log Activity (Device, OS, Browser, IP, Location)
      → Store in MongoDB → Redirect to Dashboard

2. TRAINING SESSION FLOW
   ─────────────────────
   User → Select Module → Initialize Session → Record Audio
      → Process via Web Speech API → Send to Gemini AI
      → Evaluate Response → Generate Feedback
      → Calculate Scores (Technical, Communication, Grammar, Confidence)
      → Update Progress in MongoDB → Display Results
      → Update Analytics Dashboard

3. PAYMENT FLOW
   ────────────
   User → Select Plan → Razorpay Checkout → Payment
      → Webhook Callback → Verify Signature → Update User Plan
      → Set Usage Limits → Generate Receipt PDF (Puppeteer)
      → Email Invoice → Update Subscription Table
      → Enable Pro/Enterprise Features → Redirect to Dashboard

4. RESUME BUILDER FLOW
   ───────────────────
   User → Profile Builder → Add Sections (Experience, Education, Skills, etc.)
      → Save to MongoDB (UserProfile + Relations) → Preview
      → Generate PDF (Puppeteer with HTML Template)
      → Download/Share → Public Profile URL (Optional: /u/[username])

5. ANALYTICS PIPELINE
   ──────────────────
   Session Data → Scoring Engine → Calculate Metrics:
      - Technical Score (0-10): Keyword matching, technical terms
      - Communication Score (0-10): Words per answer, clarity
      - Grammar Score (0-10): Structure, sentence formation
      - Confidence Score (0-10): Filler words, hesitation
      - Filler Word Count & Detection
      - Speaking Pace (WPM): Words per minute calculation
      - Hinglish Detection: Hindi word percentage
   → Aggregate Historical Data → Store Trends
   → Generate Charts (Recharts) → Display Dashboard
